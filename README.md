# PeopleBaseII

PeopleBaseII is a candidate-level intelligence system for voter data, campaign operations, economic context, and transparent analytics.

## Initial goals
- ingest voter registration and voter history data
- enrich with Census and BLS data
- model race probability and Democratic lean transparently
- display source tables, derived tables, and QA checks in a web app

## Stack
- Next.js
- TypeScript
- Tailwind
- PostgreSQL
- PostGIS
- Drizzle
- Netlify
- GitHub
- Cursor + ChatGPT workflow

## Current status
Project scaffold initialized. Database and ingestion work not started yet.
